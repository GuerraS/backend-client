Wirepas Backend Client
=======================

Wirepas Backend Client is a Python package meant to gather information
form a Wirepas backend and ease communication with network devices.

Requirements
------------
* Python 3.X

Installation is recommended through pypi as

.. code-block:: bash

   pip install wirepas-backend-client
